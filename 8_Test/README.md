# 8_Test — Testing & Validation ✅

Testing plans and quick checks that validate progress against Objectives and Key Results.

Purpose
- Link tests to OKRs and ensure implementations meet expected results.

What you'll find
- Small test cases for examples in `6_Symbols`.
- A testing checklist and sample commands to run local checks.

How to use
- Implement simple unit or integration checks. Record results and link failures to `7_Semblance` for post-mortem.

CLI install
- Use the root script to scaffold `8_Test/` and place test harnesses in `8_Test/harness/`.

Rendering images
- Add test-run screenshots to `assets/8_test/` to show passing/failing outputs.

Learning verb: Test it — Validate assumptions and return to reality.
